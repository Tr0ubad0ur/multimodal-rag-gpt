# FAQ

## 1. Как правильно формулировать запрос, чтобы получить лучший ответ?

Лучше работает конкретный запрос с контекстом:

- укажите цель: что именно нужно получить
- добавьте ограничения: формат, объём, стиль ответа
- при необходимости укажите, по каким файлам/папкам искать

Пример хорошего запроса:

> «Сделай краткое резюме документа в 5 пунктах и выдели риски для внедрения».

## 2. Какие типы файлов поддерживаются?

Поддерживаются:

- текст: `.txt`, `.md`
- документы: `.pdf`, `.docx`
- изображения: `.jpg`, `.jpeg`, `.png`, `.webp`
- видео: `.mp4`, `.mov`, `.avi`, `.mkv`

## 3. Есть ли ограничение на размер файла?

Да. Максимальный размер одного загружаемого файла — **50 MB**.

Если файл больше, загрузка будет отклонена.

## 4. Как загружать файлы: через кнопку или drag-and-drop?

Оба сценария допустимы:

- можно выбрать файл через кнопку загрузки
- можно перетащить файл в область вложений (drag-and-drop)

После загрузки файл индексируется и становится доступен для поиска в ответах.

## 5. Как работать с папками в базе знаний?

Обычно процесс такой:

- создайте папку (и при необходимости подпапки)
- загрузите файлы
- прикрепите файлы к нужной папке
- задавайте вопрос с фильтром по папке или по конкретным файлам

Это позволяет изолировать контекст и получать более точные ответы.

## 6. Можно ли перетаскивать сразу папку?

В большинстве сценариев загрузка идёт по файлам.

Рекомендуемый вариант:

- сначала создать структуру папок в базе знаний
- затем загрузить файлы и разложить их по папкам

## 7. Как ограничить поиск только нужными материалами?

Используйте один из фильтров:

- по папкам — если хотите искать по тематическому разделу
- по конкретным файлам — если нужен ответ из ограниченного набора источников

Чем уже область поиска, тем выше релевантность результата.

## 8. Почему ответ «слишком общий» или «не про то»?

Частые причины:

- слишком общий вопрос без контекста
- не выбраны нужные папки/файлы
- в базе знаний нет достаточных данных по теме

Что делать:

- переформулировать вопрос более конкретно
- указать формат результата (список, шаги, таблица)
- сузить поиск до нужной папки/файлов

## 9. Где посмотреть историю запросов?

Если вы авторизованы, последние запросы и ответы сохраняются в истории.

Это удобно, чтобы:

- вернуться к предыдущим ответам
- повторно использовать удачные формулировки запросов
- удалить отдельные записи при необходимости

## 10. Безопасны ли мои данные и разделяются ли они между пользователями?

Да, данные разделяются по пользователям (в авторизованном режиме).

Это значит, что:

- ваши файлы и история запросов относятся к вашему аккаунту
- поиск и ответы строятся по вашему набору данных

## 11. Что делать, если файл загрузился, но ответ его «не видит»?

Проверьте:

- файл прикреплён к нужной папке
- в запросе выбраны правильные фильтры
- формат файла поддерживается
- размер файла не превышает лимит

Если всё верно, повторите запрос с более явной ссылкой на содержание файла.

## 12. Можно ли задавать мультимодальные вопросы?

Да. Поддерживается сценарий, когда вы задаёте текстовый вопрос и добавляете вложение (например, изображение или документ), чтобы модель использовала дополнительный контекст.
